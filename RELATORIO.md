## 2. Visão Geral do Projeto
Foi desenvolvido um ecossistema composto por:
* **API em Node.js:** Responsável por realizar os cálculos de volume de piscina, custo de água, custos de materiais (hidráulica, elétrica, alvenaria/fibra/vinil), manutenção e mão-de-obra.
* **App em React:** Aplicação web contendo as telas solicitadas (Splash Screen, Login fixo, Sobre a equipe, Ajuda) e a interface de consumo e apresentação dos resultados calculados pela API.

## 3. Entregas Realizadas
* **Desenvolvimento da API:** Cada integrante foi responsável pela implementação de endpoints específicos .
* **Testes:** Foram gerados os documentos de testes unitários e funcionais. O projeto contempla cobertura de testes automatizados com Jest.
* **Integração Contínua (CI/CD):** Foi configurado o GitHub Actions para validação automatizada do código (build e testes) a cada novo push/pull request.
* **Cronograma:** Definido um cronograma de implantação de features anexado no projeto.
