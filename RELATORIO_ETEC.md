# Relatorio ETEC

## Tema

Calculos estimativos de encargos trabalhistas de empregada domestica.

## Entregas

- API Node no endpoint `/ETEC`.
- App React integrado em `/etec`.
- Testes unitarios com Jest/Supertest.
- Testes funcionais com Selenium.
- Workflow dedicado para branch `grupo4`.

## Divisao dos calculos

- Integrante 1: salario mensal.
- Integrante 2: ferias.
- Integrante 3: rescisao.

## CI/CD

A branch `grupo4` roda testes da API, build do app ETEC e e2e local no GitHub Actions. O Render oficial so refletira a ETEC depois de merge autorizado em `main`.

## Observacao

Os valores sao estimativas academicas e nao substituem sistemas oficiais ou orientacao profissional.
