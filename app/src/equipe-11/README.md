# Equipe 11 - App

Frontend React/Vite para a equipe-11.

## Variável de ambiente

Configure `VITE_API_URL` para apontar para a API da equipe-11.

Exemplo local:

```bash
VITE_API_URL=http://localhost:3001/equipe-11
```

## Build para o painel

O app é compilado com `base: '/equipe-11/'`, então o build funciona quando
publicado sob essa rota no painel principal.
